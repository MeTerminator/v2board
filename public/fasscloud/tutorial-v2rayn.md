# Windows端v2rayN使用教程

# 下载软件
浏览器打开网址 https://github.com/2dust/v2rayN/releases/latest
选择 `v2rayN-With-Core.zip` 点击并下载
![2024-08-18T06:55:58.png][1]
下载完成后解压到本地即可，绿色软件免安装，将软件文件夹放置在某个位置即可。

# 添加订阅
1. 打开 __v2rayN__ ，选择顶部 `订阅分组 -> 订阅分组设置` 
![2024-08-18T06:32:34.png][2]

2. 添加订阅， __别名__ 就是为了方便区分订阅，可根据自己需求随便填写，一般为服务提供商名称；
可选地址填入服务商提供的订阅链接，然后点击确定保存。
![2024-08-18T06:37:34.png][3]

3. 添加完成后，在主页 `订阅分组 -> 更新全部订阅（不通过代理）` 
![2024-08-18T06:38:08.png][4]

4. 至此，订阅获取完成。
![2024-08-18T06:49:57.png][5]

# 使用加速
1. 选择一条线路，右键 `设为活动服务器 (Enter)`
![2024-08-18T06:48:34.png][6]

2. 如果要 __使用__ 加速，在下方选择 `系统代理 -> 自动配置系统代理` 
如果要 __关闭__ 加速，在下方选择 `系统代理 -> 清除系统代理` 
![2024-08-18T06:51:14.png][7]

3. 检查加速是否生效
https://www.google.com/
![2024-08-18T06:53:20.png][8]
