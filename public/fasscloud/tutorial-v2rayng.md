# 安卓端v2rayNG使用教程

# 下载软件
浏览器打开网址 https://github.com/2dust/v2rayNG/releases/latest
选择 `v2rayNG_版本号_arm64-v8a.apk` 点击下载并安装
![2024-08-18T07:14:49.png][1]

# 添加订阅 & 使用加速
1. 打开 `v2rayNG` ，选择左上角菜单
![2024-08-18T07:09:49.png][2]

2. 选择 `订阅分组设置`
![2024-08-18T07:10:14.png][3]

3. 点击右上角 `+` 号
![2024-08-18T07:10:45.png][4]

4. 填入备注和订阅链接，点击右上角 `√` 保存
![2024-08-18T07:11:01.png][5]

5. 返回主页，选择右上角菜单，选择 `更新订阅`
![2024-08-18T07:11:56.png][6]

6. 选择一条线路，点击右下角按钮开始加速。
![2024-08-18T07:12:34.png][7]
